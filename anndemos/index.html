ANN Demos

On this site you will find some useful applications to aid in your understanding of how artificial neural nets work. Each of these demonstrators is designed to run in portrait mode, and is touch-enabled. To return to this page, simply use the back button on your browser.

Perceptron

Modify the weights of a perceptron and see how it changes the behavior of the network in classifying points in a 2D plane.

Multi-Layer Perceptron

Watch backpropagation in action as a multi-layer perceptron learns how to classify points in a 2D plane.

Simple Fully-Connected Network

Inspect the inner workings of a simple fully-connected MLP that has been trained on the MNIST dataset.

LeNet-5

Inspect a version of the famous LeNet-5, visualizing its learned parameters and outputs on sample images from MNIST.

LeNet-5 (on CIFAR10)

See how the LeNet-5 structure fares when applied to a very different dataset.

DNN on CIFAR10

Observe the workings of a network with new structures that enable the training of deep nets: pooling and rectified linear units (ReLU)
